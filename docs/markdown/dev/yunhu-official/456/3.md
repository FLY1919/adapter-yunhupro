# 群内消息类型控制

## 请求URL

```link
https://chat-go.jwzhd.com/open-apis/v1/group/msg-type-limit?token=016ec8e4d17a4fbda3ccb0e99b44aca0
```

## 请求类型

POST

## 请求示例

```bash
curl --location 'https://chat-go.jwzhd.com/open-apis/v1/group/msg-type-limit?token=016ec8e4d17a4fbda3ccb0e99b44aca0' \
--header 'Content-Type: application/json' \
--data '{
    "groupId": "307149245",
    "type": "text,image,video"
}'
```

## 请求头

```bash
Content-Type: "application/json; charset=utf-8"
```

## 请求参数

| 字段    | 类型   | 是否必填 | 说明                                                           |
| ------- | ------ | -------- | -------------------------------------------------------------- |
| groupId | string | 是       | 群组ID                                                         |
| type    | string | 否       | 允许的消息类型，多个类型用逗号分隔。为空表示不限制任何消息类型 |

## type参数说明

type参数为消息类型名称，多个类型用逗号分隔，如："text,image,video"

为空时表示不限制任何消息类型

## 消息类型对照表

| 类型名称   | 说明         |
| ---------- | ------------ |
| text       | 文本消息     |
| image      | 图片消息     |
| markdown   | MARKDOWN消息 |
| file       | 文件消息     |
| post       | 帖子消息     |
| expression | 表情消息     |
| html       | HTML消息     |
| video      | 视频消息     |
| audio      | 语音消息     |
| liveAudio  | 语音通话     |

## 权限说明

机器人需要在该群聊中，且拥有允许修改群信息权限（allowEditGroupInfo = 1）

## 响应内容

| 字段 | 类型   | 说明                   |
| ---- | ------ | ---------------------- |
| code | int    | 响应代码               |
| msg  | string | 响应信息，包括异常信息 |
| data | Object | 返回数据               |

## 响应示例

```json
{
    "code": -1,
    "msg": "机器人无修改群信息权限"
}
```

```json
{
    "code": 1002,
    "msg": "type格式错误，应为消息类型名称(如: text,image,video)"
}
```

```json
{
    "code": 1,
    "msg": "success"
}
```

## 错误码说明

| 错误码 | 说明                                   |
| ------ | -------------------------------------- |
| 1002   | 参数缺失或有误                         |
| 1003   | 未授权（非法token）                    |
| -1     | 服务错误（如机器人不在群中、无权限等） |

## 请求示例

### 允许多种消息类型

```json
{
    "groupId": "307149245",
    "type": "text,image,video,audio"
}
```

### 只允许文本消息

```json
{
    "groupId": "307149245",
    "type": "text"
}
```

### 不限制消息类型

```json
{
    "groupId": "307149245",
    "type": ""
}
```
