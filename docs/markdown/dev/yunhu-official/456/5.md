# 创建标签

## 请求URL

```link
https://chat-go.jwzhd.com/open-apis/v1/group/tag/create?token=016ec8e4d17a4fbda3ccb0e99b44aca0
```

## 请求类型

POST

## 请求示例

```bash
curl --location 'https://chat-go.jwzhd.com/open-apis/v1/group/tag/create?token=016ec8e4d17a4fbda3ccb0e99b44aca0' \
--header 'Content-Type: application/json' \
--data '{
    "groupId": "307149245",
    "tag": "koishi测试群",
    "color": "#FF5733",
    "desc": "测试用户标签",
    "sort": 1
}'
```

## 请求头

```bash
Content-Type: "application/json; charset=utf-8"
```

## 请求参数

| 字段    | 类型   | 是否必填 | 说明                               |
| ------- | ------ | -------- | ---------------------------------- |
| groupId | string | 是       | 群组ID                             |
| tag     | string | 是       | 标签名称，最长9个字符              |
| color   | string | 否       | 标签颜色，格式为#RRGGBB，如#FF5733 |
| desc    | string | 否       | 标签描述                           |
| sort    | int64  | 否       | 排序值，值越小越靠前               |

## 权限说明

机器人需要在该群聊中，且拥有允许控制标签组权限（allowGroupTagManage = 1）

## 响应内容

| 字段 | 类型   | 说明                   |
| ---- | ------ | ---------------------- |
| code | int    | 响应代码               |
| msg  | string | 响应信息，包括异常信息 |
| data | Object | 返回数据               |

## 响应示例

```json
{
    "code": -1,
    "msg": "颜色输入错误"
}
```

```json
{
    "code": -1,
    "msg": "标签长度过长"
}
```

```json
{
    "code": -1,
    "msg": "创建失败，该标签已存在"
}
```

```json
{
    "code": 1,
    "data": {
        "tagId": 4093
    },
    "msg": "success"
}
```

## 错误码说明

| 错误码 | 说明                                               |
| ------ | -------------------------------------------------- |
| 1002   | 参数缺失或有误                                     |
| 1003   | 未授权（非法token）                                |
| -1     | 业务错误（如机器人不在群中、无权限、标签已存在等） |
