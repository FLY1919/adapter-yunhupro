# 获取群标签列表

## 请求URL

```link
https://chat-go.jwzhd.com/open-apis/v1/group/tag/list?token=016ec8e4d17a4fbda3ccb0e99b44aca0
```

## 请求类型

POST

## 请求示例

```bash
curl --location 'https://chat-go.jwzhd.com/open-apis/v1/group/tag/list?token=016ec8e4d17a4fbda3ccb0e99b44aca0' \
--header 'Content-Type: application/json' \
--data '{
    "groupId": "307149245"
}'
```

## 请求头

```bash
Content-Type: "application/json; charset=utf-8"
```

## 请求参数

| 字段    | 类型   | 是否必填 | 说明   |
| ------- | ------ | -------- | ------ |
| groupId | string | 是       | 群组ID |

## 权限说明

机器人需要在该群聊中

## 响应内容

| 字段              | 类型   | 说明                   |
| ----------------- | ------ | ---------------------- |
| code              | int    | 响应代码               |
| msg               | string | 响应信息，包括异常信息 |
| data              | Object | 返回数据               |
| data.list         | Array  | 标签列表               |
| data.list[].tag   | string | 标签名称               |
| data.list[].desc  | string | 标签描述               |
| data.list[].color | string | 标签颜色               |
| data.list[].sort  | int64  | 排序值                 |

## 响应示例

```json
{
    "code": 1,
    "data": {
        "list": [
            {
                "tag": "koishi测试群",
                "desc": "测试用户标签",
                "color": "#FF5733",
                "sort": 1
            },
            {
                "tag": "1",
                "desc": "测试用户标签",
                "color": "#FF5733",
                "sort": 1
            }
        ]
    },
    "msg": "success"
}
```

```json
{
    "code": 1,
    "data": {
        "list": []
    },
    "msg": "success"
}
```

## 错误码说明

| 错误码 | 说明                           |
| ------ | ------------------------------ |
| 1002   | 参数缺失或有误                 |
| 1003   | 未授权（非法token）            |
| -1     | 业务错误（如机器人不在群中等） |
