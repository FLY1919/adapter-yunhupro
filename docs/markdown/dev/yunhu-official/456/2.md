# 移除群成员

## 请求URL

```link
https://chat-go.jwzhd.com/open-apis/v1/group/remove-member?token=xxxxxxxxxx
```

## 请求类型

POST

## 请求示例

```bash
curl --location --request POST 'https://chat-go.jwzhd.com/open-apis/v1/group/remove-member?token=xxxxxxxxxx' \
--header 'Content-Type: application/json' \
--data-raw '{
    "userId": "7058262",
    "groupId": "123456"
}'
```

## 请求头

```bash
Content-Type: "application/json; charset=utf-8"
```

## 请求参数

| 字段    | 类型   | 是否必填 | 说明         |
| ------- | ------ | -------- | ------------ |
| userId  | string | 是       | 被移除用户ID |
| groupId | string | 是       | 群组ID       |

## 权限说明

机器人需要在该群聊中，且拥有允许移除群成员权限（allowRemoveMember = 1）

## 响应内容

| 字段 | 类型   | 说明                   |
| ---- | ------ | ---------------------- |
| code | int    | 响应代码               |
| msg  | string | 响应信息，包括异常信息 |
| data | Object | 返回数据               |

## 响应示例

```json
{
    "code": 1,
    "msg": "success",
    "data": {}
}
```

## 错误码说明

| 错误码 | 说明                                   |
| ------ | -------------------------------------- |
| 1002   | 参数缺失或有误                         |
| 1003   | 未授权（非法token）                    |
| -1     | 业务错误（如机器人不在群中、无权限等） |
