# 用户禁言

## 请求URL

```link
https://chat-go.jwzhd.com/open-apis/v1/group/gag-member?token=016ec8e4d17a4fbda3ccb0e99b44aca0
```

## 请求类型

POST

## 请求示例

```bash
curl --location 'https://chat-go.jwzhd.com/open-apis/v1/group/gag-member?token=016ec8e4d17a4fbda3ccb0e99b44aca0' \
--header 'Content-Type: application/json' \
--data '{
    "userId": "7756242",
    "groupId": "307149245",
    "gag": 600
}'
```

## 请求头

```bash
Content-Type: "application/json; charset=utf-8"
```

## 请求参数

| 字段    | 类型   | 是否必填 | 说明                                                                                                                         |
| ------- | ------ | -------- | ---------------------------------------------------------------------------------------------------------------------------- |
| userId  | string | 是       | 被禁言用户ID                                                                                                                 |
| groupId | string | 是       | 群组ID                                                                                                                       |
| gag     | int64  | 是       | 禁言时长（秒）<br>0: 解除禁言<br>600: 禁言10分钟<br>3600: 禁言1小时<br>21600: 禁言6小时<br>43200: 禁言12小时<br>-1: 永久禁言 |

## 权限说明

机器人需要在该群聊中，且拥有允许禁言用户权限（allowGagMember = 1）

## 响应内容

| 字段 | 类型   | 说明                   |
| ---- | ------ | ---------------------- |
| code | int    | 响应代码               |
| msg  | string | 响应信息，包括异常信息 |
| data | Object | 返回数据               |

## 响应示例

```json
{
    "code": -1,
    "msg": "机器人无禁言用户权限"
}
```

```json
{
    "code": -1,
    "msg": "不可以禁言群主！"
}
```

```json
{
    "code": 1,
    "msg": "success"
}
```

## 错误码说明

| 错误码 | 说明                                   |
| ------ | -------------------------------------- |
| 1002   | 参数缺失或有误                         |
| 1003   | 未授权（非法token）                    |
| -1     | 业务错误（如机器人不在群中、无权限等） |
